# Wiki 编写标准

1. 标记使用：
  - 第一次提到非页面讲解的方块/物品/实体名称时，用` `` `包裹。
  - 使用具体代码讲解机制时，用` ```java ... ``` `包裹。
2. 物品内容：
  - 来源
  - 用途
  - 进度（若有）*这个物品可以达成哪个/哪些进度？*
  - 合成方法（若有）*哪些物品可以合成这个物品？*
  - 合成用途（若有）*这个物品可以合成哪些物品？*
  - 对应方块（若有）
    - 方块作用（若有）
    - 方块属性（若有）
3. 附魔内容：
  - 可附魔的工具
  - 附魔作用
